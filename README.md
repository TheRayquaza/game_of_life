# Game of Life (or simply as life)
This is a simple 2-dimensional cellular automaton in C

## Run the game
### Run with a board in a file
```./game_of_life yourfile```
### Run with a random seed (not supported yet)
```./game_of_life```

## Carefull : 
- You should not have a number of character different in each lines of your file,
- The number of line and character by lines should not exceed 70 characters
- There must be only appear 1 and 0 in your file
